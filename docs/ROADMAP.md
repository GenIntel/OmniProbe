# OmniProbe Roadmap

Planned work and ideas for OmniProbe.
This is a living document — items are not commitments, and contributions toward any of them are welcome (see [docs/DEVELOP.md](./DEVELOP.md)).
Missing something you'd like to see? Open a pull request or raise an issue — suggestions for new tasks, backbones, and other operations are very welcome.

## New tasks

- 3D reconstruction
- video segmentation

## New backbones

- Generative models, eg,
  - CleanDIFT
  - FLUX
  - WAN

## New operations

- Feature concatenation
- Feature upsampling
